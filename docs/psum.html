Returns the parallel sum of the input values.

Usage

psum(..., na.rm = F)

Arguments

...
input values
na.rm
a logical indicating whether missing values should be removed.

Description

Returns the parallel sum of the input values.

Examples

psum(1:5, c(1, 2, 3, 4, NA), c(1, NA, 3, 4, 5))


[1]  3 NA  9 12 NA



psum(1:5, c(1, 2, 3, 4, NA), c(1, NA, 3, 4, 5), na.rm=T)


[1]  3  4  9 12 10
